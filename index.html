Key Features

One-Click Server Setup

Automatically installs SteamCMD and configures a local Rust server for testing.

Drag & Drop Loading

Just drop your .map file into the app, and it does the rest—even if the app is not running.

Real-Time Map Monitoring

Detects map file changes instantly and updates the map list to ensure accurate data.

Desktop Notifications

Get notified when maps are saved and start a server with the newest map version by clicking the notification.

Map History Tracking

View and relaunch maps you've previously tested, no digging needed.

Server & Client Cleanup

Ensure a fresh environment with every server boot. No file mismatch errors.
